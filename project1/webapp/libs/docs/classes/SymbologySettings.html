Class SymbologySettings

A symbology-specific configuration object.

See https://docs.scandit.com/stable/c_api/symbologies.html for more details.

Note that modified ScanSettings (due to SymbologySettings changes) need to be applied to a scanner via BarcodePicker.applyScanSettings or Scanner.applyScanSettings to take effect.
